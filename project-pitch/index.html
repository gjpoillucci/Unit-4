Interactive Infographics Can Change the Way We View Data

Project Overview

Sifting through data as an individual who is only interested in the big picture is daunting. Infographics are a great way to display data in a visually appealing fashion that attracts the eye. However, somethimes one static image, or a series of images, does not due the data justice. This is where interactive infographics can come in handy. Interactive infographics allow their creators to showcasethe most important facets of knowledge, while allowing users to delve deeper into the facts as they so desire.

Project Deliverables

  • Data visuals created using GIMP, Photoshop, or a similar program
