Interactive Infographics Can Change the Way We View Data

A Project Pitch by Gwen

Sifting through data as an individual who is only interested in the big picture is daunting. Infographics are a great way to display data in a visually appealing fashion that attracts the eye.
However, somethimes one static image, or a series of images, does not due the data justice. This is where interactive infographics can come in handy.
Interactive infographics allow their creators to showcase the most important facets of knowledge, while allowing users to delve deeper into the facts as they so desire.

Project Deliverables

  • Website utilizing html and css to display the interactive infographics

  • Data visuals created using GIMP, Photoshop, or a similar program

Assignment Organization

I will need to fill the following roles to make my project a success:

  • 2 Web Developers: The Web Developers will make the interactive infographic run on the web using html and css.

  • 2 Visual Creators: The Visual Creators will design the visuals that bring the data to life.
